# Sleep Health and Lifestyle Dataset

## Overview
This dataset contains information about sleep patterns, health metrics, and lifestyle habits. It is useful for data analysis, visualization, and machine learning projects.

##  Dataset Features
The dataset includes the following columns:

- Sleep Duration
- Sleep Quality
- Physical Activity Level
- Stress Level
- Heart Rate
- Daily Steps
- BMI Category
- Age
- Gender

## Objective
The main goal of this dataset is to analyze how lifestyle and health factors affect sleep patterns.

- Health Prediction Models
- Data Visualization
